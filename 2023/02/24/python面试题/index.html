1. python计数引用

计数引用常常用在内存管理当中,当计数引用为0的时候就会释放内存地址, 如对应的操作会增加或者减少
  • 对象被创建并且赋值给某个变量， 同理删除这个变量也会减少引用
  • 变量的引用， 变量不指向该引用的时候也会减少
  • 变量被作为参数传递进函数里边，
  • 容器对象引用如list, set, str, tuple, dict， 容器被销毁或者被容器移除
